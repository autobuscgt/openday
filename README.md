## Структура проекта

Проект построен по модульному принципу. Все исходные файлы находятся в директории `/src`.

```

├── assets/               # Статические ресурсы (изображения, шрифты, иконки)

│   │   └── roads/        # SVG-схемы дорог (зеленые)
│   ├── fonts/            # Шрифты (MB.ttf, MM.ttf)
│   ├── goals/            # SVG-иконки целей/флажков (goal 1-4.svg)
│   ├── path/             # Иконки путей/маршрутов (обычные и при наведении)
│   ├── roads/            # SVG-комбинации дорог (blue_orange, green_blue, red_green)
│
├── components/           # React-компоненты
│   ├── CommonComponents/ # Переиспользуемые/Второстепенные компоненты
│   │   ├── End.js        # Экран завершения
│   │   ├── Greet.js      # Приветственный экран/модалка
│   │   ├── Modal.js      # Базовый компонент модального окна

│   ├── DevPathComponents/# Компоненты для пути разработчика
```

## Как запустить проект

### Требования
- node.js 
- npm 

### Установка и запуск

1. Клонируйте репозиторий и перейдите в папку проекта:
    git clone https://github.com/autobuscgt/openday.git
    cd openday

2. Установите зависимости:
   npm i

3. Запустите приложение в режиме разработки:
   npm start
   Приложение откроется по адресу http://localhost:3000

## Важные замечания для разработчиков

### Работа с ассетами
- **Шрифты:** закидывайте в `src/assets/fonts/` и подключайте через `@font-face` в CSS (Доступны MM - Montserrat medium, MB - Montserrat Bold).

### Контекст (Context)
Для глобального состояния квестов и путей используются `questContext.js` и `pathContext.js`.
